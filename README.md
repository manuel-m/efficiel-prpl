# efficiel-prpl

Minimal implementation of [the PRPL pattern](https://developers.google.com/web/fundamentals/performance/prpl-pattern/)

**Setup**

```
npm install
```

**Build**

```
npm run build
```

**Start**

```
// launch dtm server on localhost:8000
npm run start
```
